# The Purpose Project

A concept-stage initiative exploring how to help young people uncover a sense of purpose and translate it into action using self-discovery practices, evidence-based psychology/education, and responsible AI.

## Current status
Concept / discovery.

## Working one-liner
Help young people discover what they’re here to do—and give them a practical, supported pathway to skills, education, and paid work aligned with that purpose.

## Next 7 days (starter checklist)
- Define target population (age range, geography, barriers)
- Define outcomes (purpose clarity, training completion, job placement, retention)
- Draft program arc (assessment → plan → training → placement → mentorship)
- Pick initial pilot format (cohort size, duration, delivery)
- Create a 1–2 page donor concept note
