# Second Brain (PARA)

Private repository.

This repo follows **PARA**:
- **Projects**: time-bound outcomes with a finish line
- **Areas**: ongoing responsibilities to maintain
- **Resources**: reference material / topics of interest
- **Archive**: inactive or completed items

## Folders
- `inbox/` — quick capture; process into PARA regularly
- `projects/` — active projects (includes `the-purpose-project/`)
- `areas/` — ongoing life/work areas (health, finance, family, etc.)
- `resources/` — reference notes, learning, research
- `archive/` — completed projects and inactive notes
- `templates/` — reusable templates (meeting note, concept note, etc.)
- `ops/` — how this system works (conventions, tooling, workflows)

## Conventions (lightweight)
- Prefer lowercase-hyphen names for folders and files.
- Use ISO dates for logs: `YYYY-MM-DD.md`.
- Keep evergreen notes in `resources/` or `areas/` (whichever fits).
