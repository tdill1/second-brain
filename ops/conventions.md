# Conventions

## PARA definitions
- **Projects**: has a deliverable + finish line.
- **Areas**: ongoing standard to maintain (no finish line).
- **Resources**: reference / learning / topics.
- **Archive**: inactive.

## Capture → process
- Capture fast into `inbox/`.
- When processing, move/rename into `projects/`, `areas/`, `resources/`, or `archive/`.

## Naming
- Folders/files: `lowercase-hyphenated`.
- Logs: `YYYY-MM-DD.md`.

## Project skeleton (recommended)
Inside each project folder:
- `00-admin/` (decision-log, open-questions)
- `01-vision/`
- `02-research/`
- `03-plan/` or `03-program/`
- `04-execution/` or `04-tech/`
- `99-archive/`
